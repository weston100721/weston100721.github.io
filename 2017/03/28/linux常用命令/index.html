ssh秘钥登录

原理

使用一种被称为”公私钥”认证的方式来进行ssh登录. “公私钥”认证方式简单的解释是

  1. 首先在客户端上创建一对公私钥 （公钥文件：~/.ssh/id_rsa.pub； 私钥文件：~/.ssh/id_rsa）。
  2. 然后把公钥放到服务器上（~/.ssh/authorized_keys）, 自己保留好私钥。
  3. 当ssh登录时，ssh程序会发送私钥去和服务器上的公钥做匹配，如果匹配成功就可以登录了。
